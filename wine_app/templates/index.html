Wine Connoisseur

Welcome to Wine Connoisseur App, a django Web-app developed by Saumya Patel!

Following features are implemented as per the requirement

  • Imported csv in django model db.
  • Implemented frontend using bootstrap and a backend in the python3.6 django2.0
  • Exposed RestAPIs from the backend with wine details as json response.
  • Responsive design listing all the wine details.
  • [Plus Point]
    • Implemented feature to filter wines based on :
      • different country, province, regions.
      • Support grape variety filter search
    • Implemented the feature to sort the wines based on the price, points, country etc.
  • [Advanced Point]
    • Implemented the signup and login/logout functionality and auth based views.
